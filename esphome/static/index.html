Riden RD60xx Serial Proxy Firmware

ESPHome firmware for the ESP8266 module inside a Riden RD60xx power supply. It bridges the supply's Modbus RTU TTL link over the ESPHome native API for use with riden-modbus.

Connect the module to this computer over USB and click install. After flashing, join the fallback hotspot to provision Wi-Fi; Home Assistant generates an API encryption key when it adopts the device.
